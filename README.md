# MediaPipe - Przykłady dla Początkujących

## Wprowadzenie

To repozytorium zawiera oficjalne przykłady MediaPipe, których celem jest pokazanie podstawowych kroków potrzebnych do tworzenia aplikacji wykorzystujących naszą platformę uczenia maszynowego. Materiały zostały przygotowane specjalnie z myślą o studentach i osobach, które po raz pierwszy mają styczność z MediaPipe i uczeniem maszynowym.

## Co to jest MediaPipe Solutions?

MediaPipe Solutions to zestaw narzędzi, który usprawnia rozwój i wdrażanie uczenia maszynowego (ML) na urządzeniach. Oferuje elastyczne narzędzia low-code (mało kodu) i no-code (bez kodu), które dostarczają modularne elementy do tworzenia niestandardowych, wydajnych rozwiązań dla różnych platform.

### Główne Komponenty:

* **MediaPipe Tasks (low-code)**: Pozwala tworzyć i wdrażać niestandardowe kompleksowe rozwiązania ML od początku do końca (end-to-end)
* **MediaPipe Model Maker (low-code)**: Umożliwia tworzenie własnych modeli ML na podstawie zaawansowanych rozwiązań
* **MediaPipe Studio (no-code)**: Narzędzie do tworzenia, ewaluacji, debugowania, benchmarkowania, prototypowania i wdrażania zaawansowanych rozwiązań produkcyjnych
